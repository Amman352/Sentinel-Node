# SentinelNode

AI-powered adaptive IAM and threat orchestrator.

Monitors user login behaviour, detects anomalies using ML, and automatically responds to threats.

## Modules
- `frontend/` — Next.js dashboard
- `backend/` — Supabase edge functions + SOAR engine
- `ml-service/` — FastAPI anomaly detection service
- `docs/` — architecture docs
- `scripts/` — utility scripts
